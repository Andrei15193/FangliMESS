#1 Cod Sursa
	-> Se afla codurile sursa de la aplicatia de tip server si cea de tip client sub forma de proiect VC++ 2008

#2 Documentatie
	-> Se afla documentatia in format .docx (MS Word 2007)

#3 Executabile
	-> Se afla executabilele (executabilul pentru server si cel pentru client) impreuna cu fisierul set.txt

#4 Imagini
	-> Se afla imaginile folosite pentru realizarea proiectului in format .png si .ico.
	-> Se mai afla si fisierele .psd (PhotoShop Document)
	-> Aceste fisiere nu sunt necesare pentru rularea aplicatiilor

#5 Rulare
	-> Pentru a rula aplicatiile este necesar un sistem de operare Windows cu .NET Framework instalat
	-> Pentru a rula aplicatiile sunt necesare fisierele si/sau executabilele din directoriul "Executabile"

#6 Informatii suplimentare
	-> Se pot gasii in documentatia din folderul "Documentatie"

	-> Acest proiect a fost realizat de Fangli Andrei, toate drepturile rezervate!
